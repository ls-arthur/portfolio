O que é HTML?

HTML - Hypertext Markup Language.
HTML é uma linguagem de marcação de texto. Na internet tudo está relacionado a conteudo e links, o HTML serve para que possamos estruturar a pagina web, e utilizar a marcação de hiperlinks para conectar um conteudo a outro, caracrterizand asism essa teia de internet.

Relevante história do HTML

Até o ano de 1997 não haviam padrões para o uso da linguagem HTML, seu intuito era unicamente a disponibilização de conteudo através de hiperlinks. Isso acarretava em diversas paginas que não eram renderizadas corretamente em diferentes navegadores.
A partir de 1997 o W3C - Word Wide Web, sugiu com os primeiros padrões, os quais evoluiram até chegar no atualmente utilizado HTML5 em 2011. A W3C está no comando para definir o que se enquadra como padrões para o HTML5. Existem algumas ferramentas quem podem se tornar interessantes no desenvolvimento de paginas HTML. Por exemplo, os sites recomendados abaixo.
  • W3C - site -- Pode ser utilzado para validar se a estrutura do seu codigo html está correta.
  • Can I Use - site -- Site que mantem o controle de padrões html5 - SVG - CSS - APIs JS. Pode servir como consulta ou para verificar a compatibilidade dos navegadores com determinado recurso

Anatomia da Tag HTML

É importante entender que criar um documento html consiste em como sintaticamente e corretamente codificar uma tag html.
Gerammente uma tag html consiste no conjunto de duas tags (há excessões de apenas uma tag), onde uma representa a tag de abertura e a segunta representa a tag de fechamento. Isso serve para representar onde o conteudo da tag começa e termina, portanto é correto entender que o conteudo da tag deve ser inserido entre essas duas tags.
Uma tag de é representada pelo sinal de < >, onde dentro desses sinais existirá o elemento que identifica qual a tag desejada a se usar. Por exemplo na tag <p>, o elemento 'p' é utilizado para representar uma tag de paragrafo
As tags de aberturas são representadas pelos sinais de menor e maior juntamente com o elemento da tag. - <p>
As tags de fechamento são representadas pelos sinais de menor e maior, seguido do sinal de barra, juntamente com o elemento da tag. </p>

Cada elemento html pode possuir atributos. Os atributos sempre são inseridos na tag de abertura nunca na de fechamento. Os atributos são compostos por dois elementos, sendo eles: nome do elemento e valor do elemento. <p id="myid"> - Neste exemplo o valor de atributo "myid" é atribuido ao elemento id.

Sobre o valor dos atributos, nem sempre é nescessário cercar-los com o simbolo de "". No entanto é uma boa pratica sempre utilizar aspas simples ou aspas duplas para cercar o valor do atributo.

Semanticamente o valor de aspas simples ou duplas são equivalentes para o html, logo não existe diferença de sintaxe ou semantica em usar qualquer um dos dois.
Pode tornar-se interessante fazer uso conjungado das aspas simples e duplas quando se tem um aninhamento de valores dentro do atributo. No enteanto é importante ficar atento para que cada par de aspas feche corretamente com o seu de dentro para fora e de fora para dentro.

Estrutura basica do HTML

A estrutura do documento html é definida por suas tags principais, em um formato de blocos e que podem ser aninhadas umas as outras. desta forma temos a estrutura base sa segunte forma:

<!doctype html>
<html>
<head>
</head>
<body>
</body>
<footer>
</footer>
</html>

Vejamos o que cada bloco representa:

<!dovtype html> - Define compatibilidade da pagina com HTML5.
<html> - Abertura do bloco html que aninha todo conteudo html da pagina.
<head> - Abertura do bloco que aninhas as informações de cabeçalho da pagina.
</head> - Fechamento do bloco cabeçalho.
<body> - Abertura do bloco que representa o corpo (conteudo) de pagina.
</body> - Fechamento do bloco de conteudo.
<footer> - Abertura do bloco de rodapé da pagina.
</footer> - fechamento do bloco de rodapé.
</html> - Fechamento do bloco html.

Elementos de Titulo

Na linguagem HTML nós temos o que chamamos de elementos semanticos. Um elemento semantico é um elemento que possui algum signifaco inerente. Sendo assim podemos entender que um elemento semantico é um elemento que diz algo sobre o conteudo, pode ser sobre sua importancia, um pouco da sua descrição, basicamente sugere algo sobre seu significado.
A razão para isso é que computadores e humanos podem entender melhor uma tag semantica do que alguma tag sem signifaco algum simplesmente estrutural. Agora tomando como exemplo as tags de cabeçalho, nós temos o seguinte:
<h1> </h1>
<h2> </h2>
<h3> </h3>
<h4> </h4>
<h5> </h5>
<h6> </h6>
EM termos de titulo, podemos esperar a seguinte visualização:

Titulo do conteudo - H2

Titulo do conteudo - H3

Titulo do conteudo - H4

Titulo do conteudo - H5
Titulo do conteudo - H6
O titulo mais importante desta lista é: Titulo do conteudo - H2, em quanto que o menos importante é o: Titulo do conteudo - H6. Perceba que automaticamente o navegador atribui uma diferença na formatação da fonte entre os titulos de acordo com sua prioridade. É isto que queremos dizer quando um elemento semantico possui valor.
No entanto é importante ressaltar que mesmo que o navegador apresente uma diferença visual na exibição dos titulos, eles não devem ser utilizados para este fim. Os titulos servem para ressaltar a importancia do conteudo no documento html, e nada mais. Qualquer estilização deve ser feita através de CSS.

Referencia de entidades de caracteres HTML

Na linguagem html utilizamos caracteres especiais para escrever usa sintaxe. É comum precisar utilizar estes mesmos caracteres especiais no conteudo reproduzido na pagina, se apenas digitarmos os caracteres, o navegador os interpretará como parte da sematica html. para que isos não acnteça e posamos represetalos no conteudo utilizamos as entidades ed caracteres html.
Eis pincpais exempos abaixo:
  • Caracrete: < Representado: & lt; - (sem espaços na digitação)
  • Caracrete: > Representado: & gt; - (sem espaços na digitação)
  • Caracrete: & Representado: & amp; - (sem espaços na digitação)
  • Caracrete: © Representado: & copy; - (sem espaços na digitação)
  • Caracrete: " Representado: & quot; - (sem espaços na digitação)
  • Não quebrar espaço entre palavras; Representado: & nbsp; - (sem espaços na digitação)

Criando Links

Para criar hiperlinks em um documento html utilizamos a tag - <a> - o elemento 'a' represneta ancor = ancora. Semanticamente remete a ideia de que este link possui uma ancora com outra pagina html. Existem duas formas de apontar o link para outro documento html, uma é referenciando o arquivo html e a outra e referenciando a url da pagina html.
1ª - link interto, apontar para um arquivo html - se você tem o arquivo htm salvo em um diretorio local, você pode inserir o caminho do diretorio seguido do nome do arquivo. --- <a href="nome_arquivo.html">Conteudo do link</a>
2ª - link externo, apontar para uma url web - se você vai inserir um link para uma pagina da internet, é nescessário apontar para a url da pagina na internet.--- <a href="http://site.com">Conteudo do link</a>
Se o link da pagina for quebrado, ou o diretorio da arquivo html for modificado, a pagina não será encontrada.

Outra forma de trabalhar com links, é criando links para seções dentro da pagina html. Pa isso, utilizamos a tag de ancora <a>, e no lugar do endereço da pagina, iremos utilizar o simbolo '#' seguido do nome de link para seção. Ficando dessa forma:
<a href="#nome_seção">Conteudo do link</a>
Após isso basta atribuir o elemento de ID na tag para onde o usuário será encaminhado na pagina. Utilizar o nome de seção criado no link sem o caraqctere de '#'. E então ao clicar no link você será encaminhado para seção designada.
Esta funcionalidade pode ser exemplificada ao clicar nos links para os topicos deste documento html, localizados no inicio da pagina.

Exibindo Imagens

<img> Tag Imagem - utilizada para inserir elementos de imagem no documento html. Existem duas possibilidades no momento de inserir a imagem.
1ª - apontar para um diretorio local - se você tem o arquivo de imagem salvo em um diretorio local, você pode inserir o caminho do diretorio seguido do nome do arquivo.
2ª - apontar para um repositorio web - se você vai inserir uma imagem da internet, é nescessário apontar para a url onde a imgem está disponibilizada na internet.
Se o link da imagem for quebrado, ou o diretorio da imagem for modificado, a imagem deixa de ser exibida na pagina html.

TAGS HTML utilizadas neste documento:

  1. <!DOCTYPE html> ------ Tag que declara ao navegador a versão do HTML utlizado.

  2. <html> </html> ----------- serve para delimitar onde o conteudo html da pagina começa e onde ele termina. Toda a estrutura html deve estrar dentro desas das tags.

  3. <head> </head> ----------- Esta tag indica o inicio e o fim da parte cabeçalho na estrutura do documento html. Neste cabeçalho, algumas informações de metadados podme ser fornecidas para guiar o navegador sobre a interpretação e rederização do arquivo html. Como é o caso das tags: <meta charset> - <html lang> - <title>, as quais veremos logo asseguir.

  4. <meta charset="utf-8"> --- Esta tag informa ao navegado o tipo decodificação html que estamos utilizado.

  5. <html lang="pt-br"> ------- Tag que informa ao navegador a linguagem da pagina.

  6. <title> </title> -------------- Tag que define o titulo da pagina que será exibida no navegador.

  7. <body> </body> ------------ Body = corpo - Esta tag serve para delimitar a area em que será inserida o conteudo da pagina.

  8. <h1 - h6> </h1 - h6> ------- Tags de titulo. O conteudo desta tag é lido pelo navegador como o titulo de conteudo. A relevancia do titulo é atribuida atraves da numeração. h1 mais relevante - h6 menos relevante

  10. <ol> </ol> ------------------ order list = lista ordenada - É uma tag que cria uma lista ordenada (numerada) de itens. A tag serve apenas para Criar a lista, aind é nescessário adicionar os itens da lista através da tag <li>

  11. <ul> </ul> ------------------ unorder list = lista não ordenada - É uma tag que cria uma lista não ordenada (pontilhado) de itens. A tag serve apenas para Criar a lista, ainda é nescessário adicionar os itens da lista através da tag <li>.

  12. <li> </li> ------------------- list iten = iten de lista - É uma tag que adiciona itens dentro de listar ordenadas ou não ordenadas.

  13. <br> ------------------------- Quebra de linha = tag para fazer a quebra de linha em um texto.

  14. <p> </p> ------------------- Paragrafo - É uma tag que adiciona um campo de paragrafo dentro do html.

  15. <article> </article> ------- Artigo - Tag que adiciona um campo de artigo dentro do html.

  16. <footer> </footer> -------- Rodapé - Tag que representa a seção de rodapé do nosso arquivo html.

  17. <a> </a> -------------------- Ancora - Tag utilizada para criar a conexão de hiperlink entre um conteudo e outro. É possivel direcionar o usuário a outra pagina quando clicado no conteudo desta TAG

  18. <span> </span> ------------ Span - Tag para conteudo inline. No meu entendimento, é utilizada junto de outras tags para aplicar estilização em uma parte especifica do conteudo. A tag em si não gera valor semantico para o documento html

  19. <hr> -------------------------- Quebra tematica - Cria uma linha horizontal na pagina html. Geralmente utilizada para fazer a quebra entre um tema e outro

  20. <img> -------------------------- Imagem - tag utilizada para inserir elementos de imagem no documento html. Existem duas possibilidades no momento de inserir um arquivo de imagem, podendo ser interno ou externo.

  21. <section> </section> --------- Seção - utilziada para separar o conteudo da pagina em seções.
