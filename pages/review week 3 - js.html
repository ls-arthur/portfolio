JavaScript Basico

O que é JavaScript ?

JavaScript é uma linguagem de programação, comumente muito utilizada para o desenvolvimento, especialmente na parte de frontend.
Com ela é possível implementar itens complexos em páginas html. O JS Juntamente com HTML e CSS compõem às três principais tecnologias da Word Wide Web.

  • É uma linguagem interpretada. Isso significa que seus comandos são interpretados um a um sem a nescessidade de um compilador. No caso do JS, o navegador faz o papel de interpretador.
  • Possui um excelente controle para manipulação de comportamento em paginas web com CSS e HTML
  • É uma linguagem Orientada a objeto.
  • Trabalha com variaveis globais de Locais
  • É uma linguagem Typada Dinamicamente. Isso signivica que não é nescessário declarar o tipo das variaveis, e que uma variavel pode mudar de tipo ao longo da execussão do código.

Onde inserir o codigo JavaScript ?

Existem duas maneiras de se trabalhar com o código JS. Na primeira podemos utilizar a tag 'script' para inserir codigo JS diretamente no documento html. Mas por questões de boas praticas de desenvolvimento este metodo não é muito utilizado. A segunda forma seria referenciando um um arquivo de configuração JS externo ao arquivo documento HTML. Para fazer a referencia utilizamos o parametro ' src"diretorio/arquivo.js" ' Dessa forma temos a seguinte semantica:

<script src="js/script.js"> </script>

Definindo Variaveis, Funções e Escopo

Em JavaScript para devinir uma variavel utiliza-se a palavra reservada 'var' seguida do nome da variavel e seu valor atribuido.
var cor = "azul"
Uma função é definida utilizando-se a pafavra reservada 'function'. Em Js é possivel criar uma função iniciando o código pela palara reservada, seguido do nome da função, seu parametro e sua ação. Outra forma é criando uma variavel e atribuindo uma função a esta variavel. A semantica ficaria da segutine forma:
function soma ("parametros") {...}
var soma = function ("parametros") {...}
Dentro do codigo JS, as funções e variaveris ficam armazenadas dentro de dois escopos. 1. Escopo Global - 2. Excopo lexical
Escopo Lexical - Refere-se a um escopo onde a variavel ou função é criada e declarada dentro de uma função. Sendo assim essa variavel só existe fisicamente dentro do escopo da função.
Escopo Global - Refere-se a um escopo global onde a variavel ou função é criada é declarado no escopo global do codigo de forma a estar disponivel em todos os lugares.
O escopo em JS deverá se comportar da seguinte forma. para referencias não definidas, a variavel será procurada primeiramente dentro de seu escopo, se não encontrada, a variavel sera procurada dentro do escopo externo, se não encontrada seguirá sempre semdo buscada dentro de um excopo externo um nivel a mais acima até que por fim seja procurada dentro do escopo global. Se ainda sim não for encontrada, será definida como uma variavel indefinida.

Types JavaScript e construções de linguagem comum

JavaScript Types

Em primeiro lugar, vamos entender um pouco sobre o que sãp e quais são os Types em JS.
Um type é uma estrutura de dados particular. Types incorporados podem ser utilizados para criar outras estruturas de dados. Js possui 7 types, sendo eles 6 primitivos e 1 objeto.

  • Um tipo objeto em JS refere-se a uma coleção de pares nome/valor.
  • Um tipo primitivo em JS representa um unico e imutavel valor.
Tipos primitivos em JavaScript
  1. Boolean - corresponde a apenas dois valores: Falso e Verdadeiro.
  2. Undefined - Corresponde a um tipo o qual nenhum valor foi definido.
  3. Null - Corresponde a um valor nulo: isso significa a falta de valor.
  4. Number - Refere-se na verdade ao unico tipo numérico em JavaScript.
  5. String - Refere-se a uma sequencia de caracteres usado spara representar texto.
  6. Simbol - Em JavaScript, um valor symbol é criado através da função Symbol() , que dinamicamente produz um valor único e anônimo.

Construções de linguagem comum

Operadores em JS:
  • || - (ou) - Operador 'ou' executa comparação onde apenas um item precisa ser verdadeiro para retornar um valor True.
  • && - (e) - Operadoe 'e' executa comparação onde todos os itens devem ser verdadeiros para retornar um valor True.

A concatenação de strings pode ser utilizada através da simbologia: '+=' semanticamente ficaria da seguinte forma:
var string = "Hello"
string += World !"
console.log(string);
- - - "Hello World !" - - -

As operações matematicas em JS são muito semelhante a outras linguegens em questão de respeitar a precedencia de calculo levando em concideração a simbologia de () e []. Além disso Seus respectivos operadores por ordem de importancia são: Multiplicação (*) - Divisão (/) - Soma (+) - Subitração (-)

Em Js existem duas formas para comparação de igualdade, a regular e a estrita.

  • Igualdade regular ( == ) - Converte o type das variaveis para comparar sua igualdade de valor.
  • Igualdade estrita ( === ) - Não converte o type e verica a igualdade tanto de valor quanto de type

No JS é importante que durante a declaração de funções, na estrutura da função que o {} seja aberto na mesma linha de codigo em que a função é declarada. Neste caso não é apenas uma questão de estilo de escrita do codigo, mas sim a forma da sintaxe em que a lingugem é interpretada.
De outra forma o JS interpretará como se o comando recebesse um ';' e pulasse de linha esperando o inicio de um novo comando, o que pode acabar gerando um erro de sintaxe.

Utilizar a simbologia de ';' não é algo obrigatorio no JS. No entanto é uma boa pratica e recomendavel a utilização de ';' ao final de cada declaração.
